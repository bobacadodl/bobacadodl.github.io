EyeSpy

EyeSpy is an application that uses the computer's webcam to track eye movement in order to control the mouse completely hands-free. It was written in Java, and uses a port of the OpenCV library for image processing and eye tracking. The algorithm for eye tracking is based upon It also This project won the first place prize at HackExeter Winter 2015 (4 Pebble Steel watches)!

BookKeep

BookKeep is a textbook checkout app that I created for my school to easily keep track of textbooks. A teacher can scan a QR code or a barcode on a book, then type in a student's name to associate the book with the student. To check the book back in, the teacher simply needs to scan the barcode again. Teachers can also figure out which students have not returned books, and search the database for specific students to see what books they have checked out. The client-side uses Ionic Framework (HTML/CSS/Javascript) for cross-platform compatibility, and the server side uses PHP and a MySQL Database.

SciBowl Question Generator

This application generates Science Bowl questions for students to practice with. It allows question filtration by category, difficulty, and question type (multiple choice or short answer). The questions were collected from NSB's publicly available question PDFs by using Apache PDFbox to parse the text and dump it into a JSON file for the server to use. The application uses Node.js for the backend, Socket.io for real-time communication, and Express to serve files.
